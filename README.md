# Fraud Detection System (Java GUI)

## 📌 Description
A Java-based fraud detection system using statistical analysis (Z-score + velocity).

##  Features
- CSV-based transaction loading
- GUI for card checking
- Fraud detection using anomaly detection
- Handles Excel errors (comma, tab, missing values)

##  Tech Stack
- Java
- Swing (GUI)
- File Handling

##  How to Run
```bash
javac src/FraudDetectionSystemGUI.java
java -cp src FraudDetectionSystemGUI
